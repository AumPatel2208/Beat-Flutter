# Font Assets

This directory should contain the Courier Prime font files for screenplay formatting:

- CourierPrime-Regular.ttf
- CourierPrime-Bold.ttf
- CourierPrime-Italic.ttf
- CourierPrime-BoldItalic.ttf

You can download Courier Prime from Google Fonts:
https://fonts.google.com/specimen/Courier+Prime

Or from the original source:
https://quoteunquoteapps.com/courierprime/

Courier Prime is licensed under the SIL Open Font License (OFL).
